# Задача про календарь  

## Глоссарий  

Календарь - это [григорианский календарь](https://ru.wikipedia.org/wiki/Григорианский_календарь) после 1600 года, который содержит дни, недели и месяца года.  Основной функцией  которого является определение дня недели по дате (состоящей из номера дня и месяца).  

## Вопрос
 
Сколько всего уникальных календарей существует вне зависимости от номера года?  

*Всего существует 14 уникальных вариантов календаря:*  

- 7 возможных дней недели, с которых может начинаться год;
- 2 типа года: обычный и високосный.

Таким образом:

`7 × 2 = 14`

## Описание программы  

Пользователь вводит номер года. Программа определяет соответствующий тип календаря, получает его шаблон и выводит календарь на русском языке. 

## Данные на вход

Целое число — номер года после 1600 года. 

## Данные на выход  

Календарь указанного года, содержащий:  

- названия месяцев;  
- дни недели;  
- даты каждого месяца.  

## Требования  

Реализовать на JAVA с соблюдением принципов ООП используя любой набор известных технологий.  
## Особенности реализации  

В программе заранее создаются *14* уникальных шаблонов календаря. Годы с одинаковым типом календаря используют один и тот же объект шаблона.

Например, два обычных года, начинающихся с понедельника, используют общий шаблон, отличаясь только номером года.

Приложение разделено на несколько слоёв:  

- `domain` — доменные модели и правила их корректности;  
- `application` — фабрика шаблонов и сервис получения календаря;  
- `presentation` — консольный ввод и отображение календаря.  

## Использованные паттерны  

- *Flyweight* — хранение и повторное использование 14 общих шаблонов календаря;
- *Factory* — создание шаблонов и получение подходящего шаблона по году;
- *Strategy* — отдельная стратегия отображения календаря;
- *Dependency Injection* — передача зависимостей через конструкторы.  

## Использованные технологии
